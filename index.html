Sortierverfahren

Bubblesort

Bubblesort ist ein einfacher Sortieralgorithmus zum Sortieren von Listen und Arrays. Bei diesem Algorithmus/Sortierung wird ein Array, also eine Liste von Zahlen Schritt für Schritt sortiert, indem man die Bubble-Phasen durchführt: Man startet bei der ersten Zahl im Array. Diese wird mit der nächsten Zahl im Array verglichen und wenn diese größer ist, werden sie vertauscht. Sind sie aber in der richtigen Reihenfolge, dann bleiben sie unverändert. Anschließend wird die nächste Zahl mit der dritte im Array verglichen und so weiter, bis das Ende des Arrays erreicht ist. Dieser Vorgang wird wiederholt, bis das Array vollständig sortiert ist.(keine Zahlen sind mehr zu vertauschen) Beispiel: [4, 2, 1, 3] → [2, 4, 1, 3] → [2, 1, 4, 3] → [2, 1, 3, 4] →[1, 2, 3, 4]

Insertionsort

Insertionsort ist auch ein Algorithmus zum Sortiern von Listen und Arrays und ist für Arrays, die schon zum grossen Teil vorsortiert sind, sehr praktisch. Es sortiert eine Liste, indem er die Elemente nacheinander in einen schon sortierten Bereich einsortiert. Das erste Element wird als sortiert gesehen. Das nächste wird mit den Elementen im sortierten Bereich verglichen: Wenn das aktuelle Element kleiner ist als das im Bereich, werden die grösseren Elemente nach rechts verschoben und das aktuelle Element wird in die richtige Position eingefügt.Diesen Vorgang wird wiederholt, bis das Ende des Bereichs erreicht ist und alle Elemente der Liste sortiert sind. Die gesamte Liste ist jetzt sortiert. Beispiel: [4, 2, 1, 3] -> [2, 4, 1, 3] ->[1, 2, 4, 3]

Selectionsort

Selection Sort verfolgt das Prinzip, das kleinste oder grösste Element im unsortierten Bereich eines Arrays zu finden. und es an die richtige Position zu bringen, aber im sortieren Teil. Mit dem ersten Element des Arrays beginnt man das Sortieren und danach sucht der Algorithmus nach dem kleinsten oder größten Element.Nachdem diesen gesuchten Element gefunden wird, wird es mit dem ersten unsortierten Element im Array vertauscht. Dadurch wird das erste Element in der Liste an die richtige Position im sortierten Teil verschoben. Die Sortierung wird immer für den nächsten unsortierten Teil des Arrays wiederholt( der sortierte Abschritt wächst Schritt für Schritt, währendessen wird der unsortierte Teil kleiner),bis das Ende des Arrays erreicht ist und alle Elemente sortiert sind. Beispiel: [4, 2, 1, 3] -> [1, 2, 4, 3] ->[1, 2, 3, 4]
